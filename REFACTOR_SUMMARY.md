# 重构摘要

这份文档记录已完成的播放器和本地音乐界面重构，保留为历史说明。

## PlayerBar

- 底部播放器从高占用布局收敛为紧凑布局。
- 保留播放、上一首、下一首、播放模式、音量、喜欢、队列、桌面歌词和沉浸式歌词入口。
- 进度条、时间、封面和曲目信息在小屏下保持可用。

## LocalMusicView

- 本地音乐页支持扫描目录、刷新、列表和网格视图。
- 本地库刷新时，扫描成功的目录会按最新结果增删歌曲，扫描失败的目录会保留既有歌曲。
- 本地歌曲支持元数据编辑、封面写入和歌词写入。

## SearchView

- 搜索页分类调整为综合、单曲、歌手、专辑和歌单。
- 综合页按歌手、新专辑、歌单摘要和单曲列表组织结果。
- 顶部搜索框和搜索页搜索框支持网易云在线联想、最近搜索和当前结果联想。

## 后续维护原则

- 页面文档应跟随实际路由和文件结构更新。
- 不再保留已删除页面、服务或后端模块的说明。
- 行为变化需要配套回归脚本，并纳入 `pnpm run test:regression`。
