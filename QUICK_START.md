# 🎵 AuralFlow 快速启动指南

## 🚀 立即启动

### 开发模式（推荐）
```bash
cd auralflow
npm run tauri:dev
```

这会同时启动：
- Vite 开发服务器（热重载）
- Tauri 桌面应用窗口

### 生产构建
```bash
npm run tauri:build
```

构建产物位于 `src-tauri/target/release/bundle/`

## ✨ 核心功能一览

### 🎵 播放器
- **播放控制**: 播放/暂停/上一首/下一首
- **进度条**: 点击跳转
- **音量控制**: 拖拽或快捷键调节
- **最近播放**: 首页自动展示

### 🔍 搜索
1. 点击侧边栏"搜索"
2. 输入歌曲名或歌手
3. 点击"搜索"
4. 点击播放按钮开始播放

### 📋 播放队列
- 点击播放器右侧 **列表图标** 打开队列
- 点击歌曲切换播放
- 点击删除图标移除歌曲

### 🎤 歌词
- 点击播放器右侧 **麦克风图标** 打开歌词
- 歌词自动滚动同步
- 当前行高亮显示

### ⌨️ 快捷键
| 功能 | 快捷键 |
|------|--------|
| 播放/暂停 | `Space` |
| 上一首 | `Shift + ←` |
| 下一首 | `Shift + →` |
| 快退 5秒 | `←` |
| 快进 5秒 | `→` |
| 音量增加 | `↑` |
| 音量减少 | `↓` |
| 静音 | `M` |

### 🎨 主题切换
- 点击顶部右上角 **太阳/月亮图标**
- 或进入"设置"页面选择：
  - 浅色
  - 深色
  - 跟随系统

### 💾 系统托盘
- 关闭窗口会**最小化到托盘**（不退出）
- 左键点击托盘图标 - 显示/隐藏窗口
- 右键托盘图标 - 菜单选项
- 真正退出 - 托盘菜单选择"退出"

### ⚙️ 设置
点击侧边栏底部"设置"可配置：
- 主题偏好
- 默认音源
- 音质偏好
- 下载位置
- 查看快捷键

## 🎯 首次使用流程

### 1️⃣ 启动应用
```bash
npm run tauri:dev
```

### 2️⃣ 搜索音乐
- 点击左侧"搜索"

- 点击右下角麦克风图标

- 点击右上角主题图标
- 或进入设置页面选择

## 🐛 常见问题

### Q: 关闭窗口后应用还在运行？
A: 这是设计行为。点击关闭按钮会最小化到系统托盘，从托盘菜单选择"退出"才会真正退出。

### Q: 搜索没有结果？
A: 确保网络连接正常，音源 API 可访问。

### Q: 歌词显示"暂无歌词"？
A: 歌词功能需要配置歌词源 API，当前为预留接口。

### Q: 快捷键不生效？
A: 确保焦点不在输入框内，快捷键会在输入状态下禁用。

### Q: 如何完全退出应用？
A: 右键系统托盘图标，选择"退出"。
